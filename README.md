# Grupo Cordillera - Frontend

Frontend de la Plataforma de Monitoreo Inteligente. SPA construida con React 19 + Vite 8 + Tailwind CSS v4.

## Inicio Rápido

```bash
npm install
npm run dev
```

## Docker

```bash
docker build -t cordillera-frontend .
docker run -p 80:80 cordillera-frontend
```
